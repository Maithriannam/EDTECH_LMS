# 🚀 Learnova LMS

Learnova LMS is a modern and user-friendly Learning Management System built using PHP, MySQL, Tailwind CSS, and JavaScript.  

The platform is designed to help students learn, practice, attend live classes, submit assignments, and interact with AI-powered learning assistance — all in one place.

It includes a clean dashboard UI, responsive design, dark/light mode support, online tests, AI chatbot integration, and payment management features.

---

# ✨ Main Features

## 🎓 Student Dashboard
- Clean and modern dashboard UI
- Responsive design for all devices
- Dark / Light theme toggle
- Notifications section
- Quick navigation sidebar

---

## 📚 Courses
- Browse available courses
- Open course details

---

## 📝 Assignments
- View assignments
